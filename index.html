CRUD Todo Application

Mini Project: CRUD Todo App

    A complete JavaScript DOM Mini Project demonstrating Create, Read, Update, and Delete operations with Local Storage.

    Application Features

    Add Tasks

    Create new tasks dynamically and prevent empty submissions.

    Mark as Completed

    Toggle completed state with visual strike-through styling.

    Edit Tasks

    Update existing tasks instantly with clean UI updates.

    Delete Tasks

    Remove individual tasks safely without affecting others.

    Local Storage Support

    Persist tasks in the browser even after page reload.

    Clear All Tasks

    Reset the entire task list and clear stored data instantly.

    Technologies Used

    • HTML5 (Semantic Structure)
    • CSS3 (Modern UI Styling)
    • JavaScript (DOM Manipulation)
    • Local Storage API
